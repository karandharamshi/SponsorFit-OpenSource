# Contributing

Thanks for helping build SponsorFit.

## Principles

- User control first.
- Truthful applications only.
- Transparent scoring.
- No final auto-submit.
- Tests for meaningful changes.

## Local development

```bash
python -m venv .venv
source .venv/bin/activate
pip install -r requirements-dev.txt
make test
make lint
```

## Good first contributions

- Improve sponsorship phrase detection.
- Add sample job importers.
- Improve scoring transparency.
- Add tests.
- Improve documentation.
